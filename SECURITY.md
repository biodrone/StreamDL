# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
|  3.x.x  | :white_check_mark: |
|  2.3.x  | :white_check_mark: |
| <= 1.3  | :x:                |

## Reporting a Vulnerability

Please file an issue using the SECURITY template, giving as much detail as possible.
